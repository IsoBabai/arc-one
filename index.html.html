Getting Started
What is Arc One?
▾
Arc One is an AI-powered, non-custodial wallet interface built on Arc Network — Circle's stablecoin-native L1 blockchain. It lets you send, bridge, swap, stake, and manage USDC and other assets using plain language commands. Currently live on Arc Testnet (Chain ID: 5042002).
Arc NetworkNon-CustodialUSDC Native
How do I get started with Arc One?
▾
  • Install MetaMask browser extension (Chrome or Firefox)
  • Serve this file via HTTP — NOT by opening directly (see Troubleshooting)
  • Click Connect Wallet → approve in MetaMask
  • Arc One automatically configures Arc Testnet for you
  • Get free USDC from faucet.circle.com
  • Start sending and exploring!
Why must I use a local server instead of opening the HTML file directly?
▾
MetaMask blocks wallet injection on file:// URLs for security. You must serve over HTTP:

Node.js: npx serve . → open http://localhost:3000
Python: python3 -m http.server 8080 → open http://localhost:8080
Deploy online: Upload to Netlify or Vercel — drag and drop the HTML file and get a live URL instantly.
What wallets are supported?
▾
Any browser-injected Web3 wallet: MetaMask, Coinbase Wallet, Brave Wallet, or any wallet injecting window.ethereum. Hardware wallets like Ledger work when connected through MetaMask.
Arc Network
What is Arc Network?
▾
Arc is Circle's open Layer-1 blockchain, purpose-built for stablecoin finance. Key features:
  • USDC as native gas — predictable, dollar-denominated fees, no crypto volatility
  • Sub-second finality — deterministic block times via Malachite consensus
  • EVM compatible — works with MetaMask and all Ethereum tools
  • Institutional grade — backed by BlackRock, Visa, HSBC and 100+ institutions
What are Arc Testnet's network settings?
▾
Add Arc Testnet to MetaMask manually:
  • Network Name: Arc Testnet
  • RPC URL: https://rpc.testnet.arc.network
  • Chain ID: 5042002 (hex: 0x4cef52)
  • Currency Symbol: USDC
  • Block Explorer: https://testnet.arcscan.app
Arc One adds this automatically when you connect — no manual setup needed.
Why does Arc use USDC as gas instead of ETH?
▾
Arc is designed for stablecoin-native finance. Using USDC as the gas token means fees are always dollar-denominated and predictable — you never have to worry about ETH price volatility affecting your transaction costs. A transfer costing $0.01 will always cost $0.01, regardless of crypto market conditions.
What is the Arc block explorer?
▾
Arc Testnet uses testnet.arcscan.app as its block explorer. You can view transactions, addresses, smart contracts, and network stats. Arc One automatically links your transactions to arcscan after each confirmation.
Wallet & Connection
Does Arc One store my private keys?
▾
Absolutely not. Arc One is 100% non-custodial. Your private keys never leave MetaMask. We only read your public wallet address to display balances. We have zero access to your funds at any time.
My balance shows 0 after connecting. Why?
▾
You need testnet USDC. Open the Faucet tab in the wallet panel and visit faucet.circle.com to claim free testnet USDC. After receiving it, click the Refresh (🔄) button.
Can I use multiple wallets?
▾
Yes. Disconnect and reconnect with a different MetaMask account. Arc One automatically updates to show the new account's balance and transaction history.
Sending Funds
How do I send USDC on Arc?
▾
  • Connect your wallet on the home page
  • Use the Send form in the wallet panel
  • Enter a valid 0x... recipient address (40 hex characters after 0x)
  • Enter the amount in USDC
  • Click Est. Gas to preview the USDC fee
  • Click Send → and confirm in MetaMask
The transaction appears in History with an arcscan link once confirmed.
How much do transactions cost on Arc?
▾
Gas is paid in USDC on Arc Network. On testnet this is free test USDC with no real value. Fees are extremely low and dollar-denominated — a standard transfer costs a fraction of a cent in USDC. Arc One's Est. Gas button shows the exact cost before you confirm.
Can I cancel a transaction after sending?
▾
Arc Network has sub-second finality — transactions confirm almost instantly, making cancellation impossible after submission. Always verify the recipient address carefully before clicking Send. Use the Est. Gas preview to double-check transaction details first.
How do I view my transaction on the Arc explorer?
▾
After each confirmed transaction, Arc One shows a direct link to testnet.arcscan.app. You can also click the Explorer button in the wallet panel to see all transactions for your address.
Security
Is Arc One safe to use?
▾
Yes. Arc One is non-custodial — we never touch your funds. Every transaction requires your explicit MetaMask signature. The code is open for inspection. On testnet, all tokens have zero real value so there is no financial risk.
What data does Arc One collect?
▾
Only your public wallet address and on-chain balance — data already public on the blockchain. No email, no IP tracking, no personal data. AI interaction history is processed locally and never stored on external servers.
Can Arc One AI execute without my approval?
▾
Never. The AI can prepare, simulate, and recommend — but every on-chain action requires your explicit MetaMask signature. There is no way for Arc One to move funds without your active approval. This is a fundamental design principle.

Arc Network mainnet is planned for launch soon — Arc is currently in public testnet with 100+ institutional partners including BlackRock, Visa, and HSBC. Arc One mainnet support will follow Arc's mainnet launch. Follow our X/Twitter for updates.

Troubleshooting
MetaMask installed but wallet won't connect
▾
Almost always caused by opening via file://. Fix:
  • Run npx serve . in the folder → open http://localhost:3000
  • Or python3 -m http.server 8080 → open http://localhost:8080
  • Make sure MetaMask is unlocked (click fox icon, enter password)
  • Refresh the page after unlocking MetaMask
  • Try disabling other wallet extensions that may conflict
Transaction failed — what happened?
▾
Common causes: insufficient USDC balance (including gas), invalid recipient address, or network issues. Check the error message — Arc One shows the exact reason. Use Est. Gas before sending to verify the transaction will succeed with your current balance.
Balance not updating after receiving USDC
▾
Click the Refresh (🔄) button in the wallet panel. If still not updating, disconnect and reconnect your wallet. Arc Network has near-instant finality so balance updates should be immediate.
Arc Testnet not appearing in MetaMask
▾
Arc One adds Arc Testnet to MetaMask automatically on first connect. If it fails, add manually: Network Name: Arc Testnet, RPC: https://rpc.testnet.arc.network, Chain ID: 5042002 (hex: 0x4cef52), Symbol: USDC, Explorer: https://testnet.arcscan.app
